Sistema de E-commerce - Projeto Final POO

Este repositório contém o código-fonte da Atividade de Programação - 2026, elaborada pelo Prof. Flávio Ferro. O sistema simula um carrinho de compras para um E-commerce, aplicando regras de negócio através do paradigma orientado a objetos. A entrega final da atividade está programada para 21/07/2026[cite: 2].

Conceitos de POO Aplicados
O código foi arquitetado para exercitar e fixar os seguintes conceitos exigidos na disciplina:
- Classes Abstratas e Interfaces: Contrato base para os itens comercializáveis.
- Herança e Polimorfismo: Implementação de categorias de produtos (Eletronico e Vestuario) com comportamentos específicos na precificação.
- Overload e Overriding: Diferentes métodos de aplicar descontos (sobrecarga) e reescrita do cálculo de valor final.
- Abstração e Encapsulamento: Proteção dos atributos dos produtos, garantindo segurança na manipulação dos dados.
- Comunicação entre Objetos: A classe do carrinho interage de forma dinâmica com os itens adicionados.

 Destaques da Implementação
O sistema conta com uma interface de texto (CLI) organizada. Durante a execução, o programa imprime cabeçalhos e banners estilizados no terminal para separar visualmente:
- A inicialização do sistema e aplicação das regras de negócio (descontos e atributos).
- O fechamento do carrinho de compras com a listagem detalhada do valor final de cada item e o total a pagar.
